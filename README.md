# k11e1 (11 keys, 1 encoder)
A simple and custom 4x3 macro pad.

## Features:
- [X] **VIA Support**
- [X] **RGB Ligting**
- [X] **QMK bootmagic**

## Hardware
- **Switches**: Kailh Choc (Low Profile)
- **MCU**: Waveshare RP2040-Zero
- **Keycaps**: Chocfox PBT (Frosted Transparent Black - 1U)
