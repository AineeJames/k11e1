# k11e1 (11 keys, 1 encoder)
A custom 4x3 macro pad with full RGB lighting, a knob, and full QMK+VIA support.

## Features:
[X] **QMK/VIA Support**
[X] **RGB Ligting**

## Hardware
**Switches**: Kailh Choc (Low Profile)
**MCU**: Waveshare rp2040-zero
